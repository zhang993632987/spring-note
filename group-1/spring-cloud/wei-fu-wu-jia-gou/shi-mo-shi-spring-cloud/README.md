# 什么是 Spring Cloud

Spring Cloud 是一个工具集，它将 VMware、HashiCorp 和 Netflix 等开源公司的工作封装在交付模式中。**Spring Cloud 简化了项目的设置和配置，并为 Spring 应用程序中最常见的模式提供了解决方案**。这样我们就可以专注于编写代码，而不会陷入配置构建和部署微服务应用程序的所有基础设施的细节中。
