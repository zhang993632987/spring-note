# Spring Core

为了降低 Java 开发的复杂性，Spring 采取了以下 4 种关键策略：

* 基于 **POJO** 的轻量级和最小侵入性编程；
* 通过**依赖注入**和**面向接口**实现松耦合；
* 基于**切面**和**惯例**进行声明式编程；
* 通过**切面**和**模板**减少样板式代码。
